WHAT IS THIS?
─────────────
Physarum is a real-time multi-species slime mould simulation
running entirely in your browser using WebGL2. No install needed.

HOW TO RUN
──────────
Desktop (recommended):
  1. Open https://stevensbond.github.io/ProjectPhySim/ in Chrome, Edge, or Firefox
  2. Allow a few seconds for the simulation to initialise
  3. Use the floating control panel (top-left) to explore

  Tap the ▾ button to collapse the panel for a cleaner view.
  Use the fullscreen button for the best experience.

QUICK START
───────────
• The panel is draggable — grab the green title bar
• Collapse/expand with the ▾ button
• Species A & B are active by default (red + blue)
• Try the built-in PRESETS section for curated setups
• ↺ Respawn randomises agent positions with current settings
• ⚙ Apply & Rebuild is only needed when changing resolution
  or agent count

PERFORMANCE TIPS
────────────────
• If it runs slow, try 640×360 (360p) in Simulation Settings
  and hit Apply & Rebuild
• Reducing agents/species also helps on older hardware
• The simulation runs best in Chrome on desktop

CONTROLS AT A GLANCE
────────────────────
Presets       — load curated scenes or save your own
Species A/B/C — colour, behaviour, spawn pattern per species
Own Movement  — give each species independent movement params
               (global movement greys out when all are ticked)
Interactions  — click cells to set follow / flee / ignore
Global Movement — sensor angle, distance, turn/move speed
Visuals       — brightness and bloom glow
